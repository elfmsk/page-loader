# Page-Loader
Downloads a web-page and saves it to output dir.

## Setup

```sh
$ sudo npm install -g elfmsk-third-project
```
## Usage

```
page-loader [options] <url>

Options:
  --output [dir]  output directory (default is current)
  -h, --help      output usage information
```
